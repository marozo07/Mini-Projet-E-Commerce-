#  Mini Plateforme E-Commerce en PHP

 Description

Ce projet est une application web e-commerce développée en PHP avec PDO et MySQL.  
Il permet aux utilisateurs de :

- Consulter la liste des produits
- Voir le détail d’un produit
- Ajouter des produits au panier
- Modifier le panier
- Passer une commande
- Recevoir une confirmation de commande

L’interface est moderne, responsive et optimisée pour une expérience utilisateur fluide.



#  Fonctionnalités

 Page d'accueil
- Affichage dynamique des produits depuis la base de données
- Images produits avec gestion d’erreur (placeholder si image absente)
- Boutons :
  - Voir détails
  - Ajouter au panier
- Design en grille responsive

#Page détail produit
- Image produit
- Nom
- Prix formaté
- Description complète
- Sélection de quantité
- Ajout au panier
- Retour à la liste des produits
